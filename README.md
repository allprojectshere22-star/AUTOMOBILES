# NexGen Automobiles

NexGen Automobiles is a responsive automotive web application developed to simulate a real-world car dealership platform. The project presents new and second-hand vehicle listings through a structured, multi-page website with a clean and modern user interface.

---

## Overview

The application is designed to demonstrate frontend web development skills by creating a professional automotive website. It provides organized navigation, detailed vehicle presentation, user authentication pages, and contact information within a consistent layout.

This project serves as a foundation that can be extended into a full-stack automotive management system.

---

## Features

* Multi-page website architecture
* New vehicle showcase section
* Second-hand vehicle listings with pricing details
* User registration page
* User sign-in page
* Contact and showroom information
* Responsive design for desktop and mobile devices

## Installation

### Clone the repository

```
git clone https://github.com/your-username/nexgen-automobiles.git
```

### Navigate into the project folder

```
cd nexgen-automobiles
```

### Run the project

If it is a frontend-only project:

* Open `index.html` in a web browser
  or
* Use Live Server in Visual Studio Code

If a backend is included:

```
npm install
npm start
```

---

## Deployment

### Deploy using Vercel (GitHub Integration)

1. Push the project to GitHub
2. Log in to Vercel
3. Select "Add New Project"
4. Import the GitHub repository
5. Deploy

Vercel will automatically build and provide a production URL.
